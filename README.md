# Laravel CRM API Example

Hi, below you can find all instruction how to run this project locally.

## Deploy for the first time

Run the following commands:
- ``docker compose up -d``
- ``docker compose exec php bash``
- ``composer setup``

## Deploy onwards

- ``docker compose up -d``
- ``docker compose exec php bash``
